    Carousel Candy

    Carousel Candy is a dead simple approach to adding a slideshow/carousel to your website or application. The CSS is purposely limited as it lets the developer focus on skinning rather than overwriting prexisting styles.

    Features

    • Dragging
    • Breadcrumbs
    • Complete user controls

    Quick start

    Basic Markup

    <div class="carousel">
<div class="slide">Slide Content</div>
<div class="slide">Slide Content</div>
<div class="slide">Slide Content</div>
</div>

    Example of Generated Markup

    <div class="carousel" style="width: 800px; ">
